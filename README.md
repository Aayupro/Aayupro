### 🛠️ Key Projects

#### ⚙️ [Git-Style Version Control Engine](https://github.com/Aayupro/vcs) `C++` `Custom DS` `Myers Diff` `WAL`
* Engineered a versioned file system with branching, snapshot tagging, 3-way merging, and time-travel checkout over a version DAG without STL dependencies.
* Built custom separate-chaining hash maps and tries for $O(1)$ time-travel checkouts, Myers diff $O(ND)$ for edit scripts, binary-lifting LCA $O(\log N)$ for merge resolution, and a write-ahead log with mark-and-sweep GC.

#### 🌐 [Social Graph & Friend-Recommendation Engine](https://github.com/Aayupro/socialnet) `C++` `Graph Algorithms` `AVL Trees`
* Built an undirected social graph in STL-free C++ utilizing sorted adjacency lists and bidirectional BFS for $O(b^{d/2})$ degrees-of-separation routing.
* Ranked recommendations by mutual-friend counts via linear-time intersection and managed user feeds using self-balancing AVL trees keyed by timestamp/sequence for $O(\log N)$ inserts.

#### 🧠 [Graph RAG vs Vector RAG Comparison Engine](https://github.com/Aayupro/GraphRag) `Python` `Google GenAI` `RAG` `Graph Theory`
* Developed a custom Vector RAG (cosine similarity) and LLM-extracted Graph RAG engine featuring an inverted-index token map to optimize graph seed lookups to $O(\text{query length})$.
* Implemented multi-hop BFS traversal for configurable context depth and built deterministic fallback layers for rate-limit resilience.

#### 🎙️ [AI Interview Platform](https://github.com/Aayupro/AI-interviewer) `TypeScript` `Bun` `WebRTC` `React` `Prisma`
* Selected for **ByteRunners Hackathon Round 2**; built a voice-driven live interview system on the Bun runtime.
* Handled WebRTC SDP proxying to stream bidirectional audio over secure Gemini Live WebSockets with real-time `requestAnimationFrame` level-meter visualizations.

#### 💳 [Credit Risk & Loan Approval Engine](https://github.com/Aayupro/creditRisk) `Python` `FastAPI` `scikit-learn` `SQLite`
